Framing Hammer

HTML5 based Wire Frame development app.

Download this project as a .zip file Download this project as a tar.gz file

About

Framing Hammer was inspired by the WireFrame Tool found at http://www.grokfusebox.com/index.cfm?ref=showGoodies

The Framing Hammer allows you to quickly develop the flow of a layout by collecting three pieces of information for each page: page title, description, and exit points.

Pages are saved in the local browser using HTML5's localStorage feature.
